Purpose: Guidelines for installing a server that will be used by the networking
         testcases.

History: Nov 18 2003 - Created.

Author: Manoj Iyer manjo@austin.ibm.com

Guidelines: 
* The server should be reachable by the test machines, 
  they should be able to communicate with each other to make sure that 
  the two machines (testmachine and the server) can talk to each other.

* Install and configure the following services, as of now only these are i
  required.
    * ftp (ftpd)
    * telnet (telnetd)
    * sendmail 
    * pppoe
    * nfs (server)
    * web server (apache)

If there are any other services that is required please update the above list.
